# Duggu Optix
## Software Requirement Specification (SRS)

### 1. Project Title
Duggu Optix – AI-Powered Online Eyewear Shopping Platform

### 2. Project Overview
Duggu Optix is an online optical shopping platform that allows customers to browse, compare, and purchase eyeglasses, sunglasses, and contact lenses. The platform aims to provide a modern shopping experience with future support for AI-based virtual try-on and personalized recommendations.

### 3. Objectives
- Provide a simple and secure online eyewear shopping experience.
- Allow users to search and filter products.
- Enable online ordering and order tracking.
- Provide an admin panel to manage products and orders.
- Integrate AI-based features in future versions.

### 4. User Roles
1. Customer
2. Admin

### 5. Technology Stack
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Python Flask
- MySQL
- Git & GitHub

### 6. Future AI Features
- Virtual Try-On
- Face Shape Detection
- Smart Product Recommendations
